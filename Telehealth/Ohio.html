How does the state define telemedicine?    
Law: "The practice of telemedicine" means the practice of medicine in this state through the use of any communication, including oral, written, or electronic communication, by a physician located outside this state.
Medicaid: "Telemedicine" is the direct delivery of services to a patient via synchronous, interactive, real-time electronic communication that comprises both audio and video elements.[1][2]

 

Practice Standards and Licensure

Question Answer Clarifying Information
Must a provider establish a relationship with a patient prior to engaging with them via telemedicine? No Reference Found[2]  
Must a telepresenter or health care provider be present with a patient during a telemedicine encounter? No Reference Found[2]  
Must providers obtain informed consent from a patient prior to a telemedicine encounter? Yes[2]  
If applicable, what type of consent is required? Not Sure[2] Unspecified type of consent required.
Can telemedicine providers from bordering states legally conduct telemedicine visits with patients in this state? Yes[1] OH does not offer border reciprocity but out-of-state providers can apply for an Ohio telemedicine certificate.
Has this state enacted laws adopting the Federal State Medical Boards (FSMB) compact which enforces expedited licensing for out-of-state practices? No[4]  
 

Medicaid Coverage and Reimbursement

Question Answer Clarifying Information
Is there state legislation regarding Medicaid reimbursement for telemedicine services? Yes[2]  
Does the state Medicaid program reimburse for services rendered via telemedicine? Yes[2]  
Does the state Medicaid program reimburse telemedicine services at the same rate as services delivered in-person? Yes[2][3] The rate schedule that is posted on the state website does not distinguish between services that are provided in-person and via telemedicine.

Medicaid Service Limitations

Question Answer Clarifying Information
Does the state Medicaid program apply any geographic restrictions to telemedicine encounters? Yes[6] The distant and originating sites must be at least 5 miles apart.
Does the state Medicaid program limit coverage and reimbursement for telemedicine to a prescribed set of health conditions or services? Yes[6] Eligible services:
  • Evaluation and management services
  • Psychiatric services
Does the state Medicaid program limit the frequency with which they will reimburse for telemedicine services for a given patient? No Reference Found[6]  
Does the state Medicaid plan restrict or provide exceptions for services provided via telemedicine for end-stage renal disease or dialysis? (e.g., in-person check-ups between treatments) No Reference Found[6]  

Medicaid-Eligible Patient Settings

Question Answer Clarifying Information
Does the state limit Medicaid reimbursement for telemedicine services based on patient setting (e.g., where the patient is located at the time of a telemedicine encounter)? Yes[6] Eligible originating sites:
  • Primary Care Clinic
  • Outpatient Hospital
  • Rural Health Clinic – Medical
  • Federally Qualified Health Center* – Medical
  • Professional Medical Group
*When the rendering site is a federally qualified health center the rendering practitioner must be a medical doctor, doctor of osteopathic medicine or licensed psychologist.

Medicaid-Eligible Provider Type

Question Answer Clarifying Information
Does the state limit Medicaid reimbursement for telemedicine services based on the type of provider facilitating the telemedicine encounter? Yes[6] Eligible providers:
  • Physician
  • Podiatrist
  • Optometrist
  • Psychologist
Note: When the rendering site is a federally qualified health center the rendering practitioner must be a medical doctor, doctor of osteopathic medicine or licensed psychologist.

Medicaid-Eligible Technologies

Question Answer Clarifying Information
Does the state Medicaid program reimburse for telemedicine services provided via live video conference? Yes[2]  
Does the state Medicaid program reimburse for telemedicine services provided via store and forward technologies? No[2] "Telemedicine" is the direct delivery of services to a patient via synchronous, interactive, real-time electronic communication that comprises both audio and video elements.
Does the state Medicaid program reimburse for telemedicine services provided via remote patient monitoring? No Reference Found[2]  
Does the state Medicaid program reimburse for telemedicine services provided via email/phone/fax? No[2]  
Does the state Medicaid program reimburse for synchronous telemedicine encounters? Yes[2]  
Does the state Medicaid program reimburse for asynchronous telemedicine encounters? No[2]

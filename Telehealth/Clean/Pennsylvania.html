How does the state define telemedicine?    
Telemedicine is the use of real-time interactive telecommunications technology that includes, at a minimum, audio and video equipment as a mode of delivering consultation services.
Telepsych (telepsychiatry and telepsychology) is the use of electronic communication and information technologies to provide or support clinical psychiatric and psychological care at a distance.[1]

 

Practice Standards and Licensure

Question Answer Clarifying Information
Must a provider establish a relationship with a patient prior to engaging with them via telemedicine? No[4]  
Must a telepresenter or health care provider be present with a patient during a telemedicine encounter? Yes[4]  
Must providers obtain informed consent from a patient prior to a telemedicine encounter? No[5] Exception: Written consent is required for the delivery of telepsych services.
If applicable, what type of consent is required? N/A[5]  
Can telemedicine providers from bordering states legally conduct telemedicine visits with patients in this state? No Reference Found[4]  
Has this state enacted laws adopting the Federal State Medical Boards (FSMB) compact which enforces expedited licensing for out-of-state practices? Yes[1] Implementation delayed
 

Medicaid Coverage and Reimbursement

Question Answer Clarifying Information
Is there state legislation regarding Medicaid reimbursement for telemedicine services? No PA has not yet passed legislation, however the PA legislature is considering a bill (introduced June 2017) that aims to make telemedicine reimbursement equal to in-person care.
Does the state Medicaid program reimburse for services rendered via telemedicine? Yes[4]  
Does the state Medicaid program reimburse telemedicine services at the same rate as services delivered in-person? No Reference Found[4]  

Medicaid Service Limitations

Question Answer Clarifying Information
Does the state Medicaid program apply any geographic restrictions to telemedicine encounters? No[4] However, Medicaid recommends parameters that might necessitate and/or justify the use of telehealth.
Does the state Medicaid program limit coverage and reimbursement for telemedicine to a prescribed set of health conditions or services? No[4][5] Exception: No limitations for general telehealth services, but telepsych services are limited to:
  • Psychiatric diagnostic evaluations
  • Psychological Evaluations
Does the state Medicaid program limit the frequency with which they will reimburse for telemedicine services for a given patient? No[2]  
Does the state Medicaid plan restrict or provide exceptions for services provided via telemedicine for end-stage renal disease or dialysis? (e.g., in-person check-ups between treatments) No Reference Found[4]  

Medicaid-Eligible Patient Settings

Question Answer Clarifying Information
Does the state limit Medicaid reimbursement for telemedicine services based on patient setting (e.g., where the patient is located at the time of a telemedicine encounter)? No[4]

Pennsylvania does not explicitly define eligible patient settings/originating sites, but does require patient consultation to occur at an "enrolled site." Therefore, it appears any Medicaid enrolled site may be eligible for telemedicine reimbursement.

Note: Home is a recognized site of care for monitoring services delivered to qualified members aged 60+ as part of the TeleCare program, which was established through a Home and Community-Based Services Waiver."

Medicaid-Eligible Provider Settings

Question Answer Clarifying Information
Does the state limit Medicaid reimbursement for telemedicine services based on the type of provider facilitating the telemedicine encounter? Yes[4][5] Eligible providers include:
  • Physicians
  • Certified registered nurse practitioners
  • Certified nurse midwives
  • Psychologists (for telepsych services only)

Medicaid-Eligible Technologies

Question Answer Clarifying Information
Does the state Medicaid program reimburse for telemedicine services provided via live video conference? Yes[4]  
Does the state Medicaid program reimburse for telemedicine services provided via store and forward technologies? No[4]  
Does the state Medicaid program reimburse for telemedicine services provided via remote patient monitoring? No[4]  
Does the state Medicaid program reimburse for telemedicine services provided via email/phone/fax? No[4]  
Does the state Medicaid program reimburse for synchronous telemedicine encounters? Yes[4]  
Does the state Medicaid program reimburse for asynchronous telemedicine encounters? No[4]
